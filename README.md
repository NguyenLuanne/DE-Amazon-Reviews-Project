# DE-Amazon-Reviews-Project
Create an end-to-end dataflow for Amazon Reviews dataset to execute an NLP task.

## Get data
Use git lfs to download the dataset from the repository.
- Set up git lfs
```bash
    git lfs install
```
- Get data
```bash
    git lfs fetch
```
- Replace pointers with actual data
```bash
    git lfs checkout
```
